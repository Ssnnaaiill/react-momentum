## 프로젝트 준비
CSS를 좀 더 쉽게 적용하기 위해 yarn을 사용하여 `styled-components`, `styled-reset` 을 설치함니다. 또한, data type check를 위해 `prop-types` 도 설치합니다.
```
yarn add styled-components styled-reset prop-types
```

다음과 같은 명령어를 사용해서 App을 실행합니다.
```
yarn start
```
